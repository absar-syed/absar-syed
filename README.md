## 🤖 Featured Project — CoursewiseAI

<div align="center">

> **An AI-powered study tool that turns your course materials into summaries, flashcards, quizzes, and Q&A — grounded strictly in your own documents.**

</div>

```
📄 Upload PDF / PPTX / DOCX / MD
         ↓
🔪 LlamaIndex chunks + indexes your documents
         ↓
🧠 Gemini Embeddings → stored in Supabase pgvector
         ↓
🔍 Cosine similarity retrieval (with refusal logic)
         ↓
⚡ Groq (Llama 3.3-70B) generates grounded responses
         ↓
📝 Summaries • Flashcards • Quizzes • Q&A + Source Citations
```

| Layer | Technology |
|---|---|
| **Frontend** | Next.js 14, React 18 |
| **Backend** | Python 3.11, FastAPI, LlamaIndex, Pydantic v2 |
| **LLM Inference** | Groq — Llama 3.3-70B-Versatile |
| **Embeddings** | Google Gemini `gemini-embedding-001` |
| **Vector Store** | Supabase PostgreSQL + pgvector |
| **Auth** | Google OAuth 2.0 via Authlib |
| **Infrastructure** | Google Cloud Run, Artifact Registry, Cloud Build CI/CD |
